    -> Python 

        -> Python is a high-level, general-purpose programming language. 
        It emphasizes code readability, simplicity, and ease-of-writing 
        with the use of significant indentation, "plain English" naming, 
        an extensive ("batteries-included") standard library, and garbage collection.


        -> Python is a high-level programming language. 

            -> High Level Languages

                -> High Level languages means "Human Understandable" language (English)

                -> Python Code Looks Like English

                    num1 = 1.5
                    num2 = 2.5
                    sum = num1 + num2 
            
            -> Low Level Languages

                -> Low Level Languages means "Machine Understandable" language 
                (Binary Bits i.e 010101010101)

                -> 01110000 01111001 01110100 01101000 01101111 01101110 -> python

        -> Python is Dynamically Typed Checked & Garbage collected 


            -> Dynamically Typed Checked Language

                -> We have a concept called "Data Types"

                    -> Data Types: Categorization Of Data 

                        -> Text, Whole Numbers, Decimal Numbers etc 

                -> NOTE: Dynamically Typed Checked Means, we don't need to Explicitly
                specify Data Types 

                    -> product = "iphone"
                    -> price = 120000
                    -> rating = 4.6 
                    -> etc 


            -> Static Typed Checked Language

                -> NOTE: Static Typed Checked Means, we need to Explicitly
                specify Data Types

                    -> String product = "iphone"
                    -> int price = 120000
                    -> double rating = 4.6 
                    -> etc

        -> Python is Garbage collected 

            -> Garbage collected means Automatically remove "Unused" Data 

                -> Python Automatically Cleans Up "Unused" Data from Memory / RAM 

                    product = "iphone"
                    price = 120000
                    rating = 4.6

                    print(product)
                    print(price)

                -> Python Cleans Up rating Memory as it's never used in the program

        
        -> Python is an Interpreted Language

            -> An Interpreted Language Executes Code Line By Line 

        -> Java is Compiled Language 

            -> A Compiled Language Translates the code into Byte Code 
            and then Executes all the code at once 

            -> NOTE: Performance wise Compiled Languages are FASTER
